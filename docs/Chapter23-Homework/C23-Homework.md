## 原始数据优化

```json

  "instruction": "你现在是一位专业的医疗问诊助手。请根据下面患者的个人信息和症状描述，通过清晰的逻辑推理，给出初步的病情评估、生活建议以及后续的就医指导。",
  "input": "男，28岁，程序员，最近一周每天工作到半夜，感觉头晕、脖子疼，有时候还恶心。",
  "output": "..."
}
```

### output 字段完整内容

**思维链（CoT）部分：**

```
<think>
1. 提取关键信息：男，28岁，程序员（长期伏案工作）。
2. 诱因与症状：近期连续一周熬夜；症状为头晕、颈痛、偶发恶心。
3. 病理分析与推理：
   - 颈椎因素：程序员是颈椎病高发人群。长期低头导致颈部肌肉劳损，近期高强度熬夜可能诱发了
     交感神经型或椎动脉型颈椎病发作，压迫导致脑供血不足，引发头晕和恶心。
   - 疲劳因素：连续半夜下班导致严重睡眠不足，植物神经功能紊乱，同样会产生头晕、恶心等躯体症状。
4. 制定回复结构：先进行专业但克制的症状分析，接着给出日常缓解的实操建议，
   最后必须加上负责任的就医提醒和 AI 免责声明。
</think>
```

**最终回答（Response）部分：**

您好！根据您描述的症状，您的身体可能在向您发出严重疲劳和颈椎受损的警告。以下是初步的分析与建议：

**1. 症状初步评估**

作为程序员，您的颈部长期处于紧绷状态，很容易出现颈部肌肉劳损。最近一周连续熬夜加剧了身体透支，极有可能是诱发了**颈椎退行性反应（疑似颈椎病发作）**或**过度疲劳导致的植物神经紊乱**。颈椎压迫或脑部供血不足都会直接引起头晕和恶心的症状。

**2. 日常缓解建议**

- **紧急休整**：请务必停止熬夜，今晚保证充足的睡眠，这是缓解头晕恶心最有效的方法。
- **改善姿态**：工作时调整显示器高度与视线平齐，每隔 40 分钟起身活动一下肩颈。
- **物理缓解**：可以用温热的毛巾敷在脖子后方，缓解肌肉痉挛。

**3. 专业就医指导**

如果充分休息后，头晕、恶心的症状依然没有明显缓解，或者开始出现手臂发麻等情况，请尽快前往医院的**骨科或脊柱外科**就诊，进行颈椎 X 光或核磁共振排查。

> *注：我是 AI 医疗问诊助手，以上分析仅供参考，不能替代专业医生的面诊与处方。*

## 优化思路小结

| 优化维度 | 说明 |
|----------|------|
| **指令（Instruction）** | 改得更加具体，明确告诉模型要扮演的角色以及需要输出的具体板块（病情评估、生活建议、就医指导）。 |
| **思维链（CoT）** | 在 `output` 中加入了 `<think>` 标签，强迫模型先拆解患者特征，再推导病理原因，最后规划回答结构。这也符合课程中强调的"复杂性"原则，即包含多步推理逻辑。 |
| **最终回答（Response）** | 去掉了之前绝对化且单薄的"颈椎病"三个字，换成了翔实的分析，并补充了实用的建议和不可或缺的免责声明，这样输出质量（Quality）才算达标。 |
